# RedsecStats

## What is it
Mini app for getting revive statistics in Battlefield 6 RedSec.
The app uses [Gametools API](https://api.gametools.network/docs) for getting your RedSec statistic. 

## How to use
1) Run the app
2) Select one of available platforms from the list. Enter just a digit
3) Enter your username (PSN, XBL, etc.)
4) Wait for information to be received
